# A3_2024.2-Docker

## Passos para rodar a aplicação:

- Clone o repositório no diretório desejado
```
git clone https://github.com/deveju/A3_2024.2-Docker.git
```

- Navegue até o repositório clonado
```
cd A3_2024.2-Docker
```

- Suba os serviços:
```
docker-compose up
```
